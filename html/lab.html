Architecture Deep Dive

Domain Controller

The backbone of the lab is a Windows Server 2022 Domain Controller. I configured realistic organizational units (OUs) reflecting a typical corporate hierarchy (HR, IT, Sales).

Key configurations include strictly defined Group Policy Objects (GPOs) to disable Windows Defender on victim machines (for testing) and enforce audit logging for SIEM ingestion.

Kerberos LDAP DNS/DHCP GPO Hardening

Firewall

OPNSense acts as the gateway between the WAN and the isolated lab network. It is configured with strict firewall rules to allow only necessary outbound traffic while blocking all non-VPN inbound connections.

I enabled Suricata IPS in promiscuous mode to inspect lateral movement traffic between the User VLAN and the Server VLAN.

VLAN Segmentation Suricata IPS Deep Packet Inspection

SIEM Solution

Wazuh is deployed to collect and correlate logs from all endpoints. Wazuh Agents are installed on both Windows 10 and Ubuntu clients.

I created custom decoders to detect specific attack vectors such as Mimikatz execution and PsExec lateral movement. The dashboard provides real-time visibility into the security posture.
